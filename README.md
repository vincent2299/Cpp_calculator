# C++ Console Calculator
A simple command-line calculator built with modern C++ to practice my C++ skills and demonstrate object-oriented programming principles. 

## Features
* Object-Oriented Design (OOD)
* Separation of concerns (Header and Implementation files)
* Exception handling for division by zero and invalid inputs

## How to Run
Compile the source files using any standard C++ compiler:
`g++ -o calculator main.cpp Calculator.cpp`
